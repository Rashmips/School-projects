Abstract

  • DATA SOURCE
    • Collected from 1990 US Census, 1995 US FBI Uniform Crime Report, 1990 US Law Enforcement Management and Administrative Statistics Survey, available from ICPSR at U of Michigan.
    • Posted to UCI archive by creator Michael Redmond
    • http://archive.ics.uci.edu/ml/datasets/Communities+and+Crime+Unnormalized

  • DATA DESCRIPTION: Violent crimes per 100k population for US counties (dependent variable) and several socio-economic statistics of those counties (predictors)

  • AIMS - Use data to analyze correlation between violent crimes (dependent variable) and race, income, unemployment rate and police presence (independent variables) in several. Ultimately use the data to devise a predictive formula for violent crime rates.

  • STATISTICAL ANALYSIS - performed using Rstudio

Variable Descriptions

Checking Non-Constant Variance

For the following fitted linear model, the vertical width of the scatterplot increases across the fitted values indicating that the variance in the error terms is not constant.

Checking Non-Constant Variance using Transformation of Data

The goodness of fit measure, r-squared, increases to .631 when taking the log of the data

When transforming the data by fitting the linear models with the log values of the predictors, the residual plot shows that the points are randomly scattered around zero, indicating that error terms have a mean near zero. The vertical width of the scatter doesnt appear to increase or decrease across the fitted values, so we assume that the variance in the error terms is constant.

fit <-  lm(log(perCrimes) ~ region + pctWhite + pctblack + pctAsian + pctHisp + medInc + pctUnemp + perPolice, data=Crime)
